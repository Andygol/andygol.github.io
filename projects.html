Projects

Istio

Istio is an open platform for connecting, managing, and securing microservices, making it easy to build secure and reliable cloud workloads, with or without sidecars. I have localized the Istio mesh service documentation into Ukrainian, starting with version 1.24.

Helm

Helm is a package manager for Kubernetes, enabling developers and operators to package, configure, and deploy applications on Kubernetes clusters easily. I made and maintain Ukrainian localization of Helm documentation.

Kubernetes Documentation Kubernetes Docs

Kubernetes, also known as K8s, is an open source system for automating deployment, scaling, and management of containerized applications. I made and maintain Ukrainian localization of Kubernetes documentation.

Welcome to OpenStreetMap Welcome to OSM

Modernization of the welcome mat of the OpenStreetMap project. This revision is aimed at introducing multilingual support, simplifying search, and adding a toggle between day and night modes.

Switch2OSM

A multilingual version of the Switch2OSM project aimed at helping people get started using OSM data in their own projects. It uses Material for MkDocs for multilingual support, night and day modes, and search.

OpenStreetMap Ukraine OSM Ukraine

The website of the Ukrainian community of the OpenStreetMap project, describing the goals and principles of the project, which also includes a map created by the community.

OpenStreetMap OSM

Ukrainian localization of the OpenStreetMap project website .

OpenStreetMap Wiki OSM Wiki

Creation of Ukrainian versions of the OpenStreetMap project documentation .

iD editor iD editor

Creation of a Ukrainian version of the iD editor user interface, help, and walkthrough.

JOSM JOSM

I spearheaded the development of the Ukrainian version of the JOSM editor user interface and accompanying documentation. I also provided guidance to maintainers on implementing improvements and adding new functionality to the editor, as well as creating plugins and validation rules.

HOT Tasking Manager HOT TM

The localization of the Tasking Manager user interface is a key aspect of the administration of projects undertaken by the Ukrainian OpenStreetMap (OSM) community.

MapRoulette MapRoulette

The localization of the MapRoulette user interface and the administration of projects undertaken by the Ukrainian OpenStreetMap community. MapRoulette has dozens of mapping Challenges that are created by experienced mappers.
